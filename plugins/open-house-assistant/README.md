# Open House Assistant

> Everything you need before, during, and after every open house.

## What It Does

- **Pre-event**: Marketing posts, email blasts, flyers, prep checklists, talking points
- **During**: Qualifying questions, conversation starters, notes template
- **Post-event**: Personalized follow-up for every visitor type + seller feedback report

## Setup (5 minutes)

**Claude**: Create Project → paste `claude/system-prompt.md` → start chatting
**ChatGPT**: Create GPT → paste `codex/instructions.md` → configure from `codex/config.json`

## Platforms

### Claude Code
Copy `claude/system-prompt.md` into your Claude project instructions.
Add the MCP server to `.mcp.json` for live CRM/software integration.

### ChatGPT / Codex
Create a Custom GPT using `codex/instructions.md` as the system prompt.
Upload knowledge files from the `knowledge/` folder.

### Perplexity
Use `perplexity/system-prompt.md` as your Perplexity Space instructions.

### Manus
Use `manus/instructions.md` as your Manus agent instructions.

# Listing Description Templates by Property Type
> Fill-in-the-blank MLS descriptions and marketing copy. Each template is fair housing compliant and optimized for search.

## Fair Housing Language Guidelines

### NEVER Use
- "Perfect for families" / "ideal for couples" / "great for young professionals"
- "Family-friendly neighborhood" / "quiet community" / "safe area"
- References to nearby churches, synagogues, mosques, or temples
- "Walking distance to [religious institution]"
- "Master bedroom" (use "primary bedroom" instead)
- "His and hers" (use "dual" or "double")
- "Handicap accessible" (use "accessible features")
- Language implying who should or shouldn't live there

### ALWAYS Use
- Property feature descriptions (the house itself, not who should live in it)
- Measurable details (square footage, room count, lot size)
- "Primary bedroom" instead of "master bedroom"
- "Accessible" instead of "handicap"
- Neighborhood amenities described factually (parks, schools, shopping — without implying demographics)

## Template 1: Starter Home (Under $350K)

### MLS Description (~250 words)
```
Welcome to [Address], a [beds]-bedroom, [baths]-bathroom home offering 
[sqft] square feet of well-designed living space in [Neighborhood]. Built 
in [year], this home features [top feature #1 — e.g., "an updated kitchen 
with granite countertops"], [top feature #2], and [top feature #3].

Step inside to find [describe entry/living area — e.g., "an open-concept 
living area with natural light throughout"]. The kitchen includes [kitchen 
details — appliances, countertops, storage]. [Beds] bedrooms provide 
comfortable living space, with the primary bedroom offering [primary BR 
detail — closet, ensuite, size].

Outside, enjoy [outdoor features — e.g., "a fully fenced backyard with a 
patio" or "a low-maintenance yard"]. Additional features include [3-5 
extras: garage, new roof, HVAC, water heater, flooring, etc.].

Located in [Neighborhood/area] with convenient access to [2-3 nearby 
amenities — shopping, dining, parks, transit]. 

Don't miss this opportunity at $[price]. Schedule your showing today.

[Agent Name] | [Brokerage] | [Phone]
```

### Power Phrases for Starter Homes
- "Move-in ready" / "Turnkey condition"
- "Outstanding value" / "Priced to sell"
- "Incredible opportunity" / "Rare find at this price point"
- "Low-maintenance living" / "Easy commute to [area]"

## Quick Reference: Power Words by Emotion

| Emotion | Words |
|---------|-------|
| **Urgency** | Won't last, rare find, just listed, back on market, price improvement |
| **Value** | Outstanding value, below market, priced to sell, turnkey, move-in ready |
| **Luxury** | Bespoke, curated, resort-caliber, artisan, architectural, exquisite |
| **Warmth** | Inviting, sun-drenched, welcoming, charming, character |
| **Space** | Expansive, generous, open-concept, soaring, airy |
| **Quality** | Custom, premium, upgraded, designer, high-end, professional-grade |
| **Location** | Sought-after, established, tree-lined, walkable, convenient |

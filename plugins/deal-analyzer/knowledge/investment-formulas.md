# Real Estate Investment Formulas & Benchmarks
> Complete reference for property investment analysis. All formulas with worked examples.

## Core Formulas

### Net Operating Income (NOI)
```
NOI = Gross Rental Income - Vacancy Loss - Operating Expenses

Where Operating Expenses include:
- Property taxes
- Insurance
- Maintenance/repairs
- Property management fees
- Utilities (if landlord-paid)
- Landscaping
- HOA fees
- Accounting/legal
- Advertising/vacancy costs

Operating Expenses EXCLUDE:
- Mortgage payments (debt service)
- Depreciation
- Capital expenditures
- Income taxes
```

### Cap Rate (Capitalization Rate)
```
Cap Rate = NOI / Purchase Price × 100

Example:
  NOI = $36,000/year
  Purchase Price = $450,000
  Cap Rate = 36,000 / 450,000 = 8.0%

Benchmarks by property type:
  Single Family Rental:   5-8%
  Small Multifamily (2-4): 6-10%
  Apartment Complex (5+):  5-8%
  Commercial Retail:       6-12%
  Office:                  6-10%
  Industrial:              5-9%
```

### Cash-on-Cash Return (CoC)
```
CoC = Annual Pre-Tax Cash Flow / Total Cash Invested × 100

Annual Cash Flow = NOI - Annual Debt Service
Total Cash Invested = Down Payment + Closing Costs + Rehab Costs

Example:
  NOI = $36,000/year
  Annual Mortgage = $24,000/year
  Cash Flow = $12,000/year
  Down Payment = $90,000
  Closing Costs = $11,250
  Total Invested = $101,250
  CoC = 12,000 / 101,250 = 11.9%

Benchmarks:
  Excellent: > 12%
  Good:      8-12%
  Acceptable: 5-8%
  Poor:       < 5%
```

### Gross Rent Multiplier (GRM)
```
GRM = Purchase Price / Annual Gross Rent

Example:
  Price = $450,000
  Monthly Rent = $4,400 (4 units × $1,100)
  Annual Rent = $52,800
  GRM = 450,000 / 52,800 = 8.5

Benchmarks:
  Excellent:  < 8
  Good:       8-12
  Average:    12-15
  Expensive:  > 15
```

### Debt Service Coverage Ratio (DSCR)
```
DSCR = NOI / Annual Debt Service

Example:
  NOI = $36,000
  Annual Mortgage = $24,000
  DSCR = 36,000 / 24,000 = 1.50

Benchmarks:
  Lender minimum: 1.20-1.25
  Comfortable:    1.30-1.50
  Strong:         > 1.50
  Risky:          < 1.20
```

### Break-Even Occupancy
```
Break-Even Occ = (Operating Expenses + Debt Service) / Gross Potential Rent × 100

Example:
  Expenses = $16,800/year
  Debt Service = $24,000/year
  Gross Rent = $52,800/year
  Break-Even = (16,800 + 24,000) / 52,800 = 77.3%

Benchmarks:
  Safe:    < 80%
  Caution: 80-90%
  Risky:   > 90%
```

### 1% Rule (Quick Screen)
```
Monthly Rent / Purchase Price ≥ 1%

Example:
  Price = $200,000
  Monthly Rent = $2,000
  Ratio = 2,000 / 200,000 = 1.0% ✓ Passes

Note: Harder to achieve in expensive markets.
  0.8% or higher is often acceptable in appreciating markets.
```

### 50% Rule (Expense Estimate)
```
Operating Expenses ≈ 50% of Gross Rental Income

Use this for quick back-of-napkin analysis when you don't have expense details.
NOI ≈ Gross Rent × 50%
```

### Internal Rate of Return (IRR) — 5 Year Hold
```
IRR accounts for:
  Year 0: Initial investment (negative cash flow)
  Years 1-5: Annual cash flow
  Year 5: Sale proceeds (appreciation + mortgage paydown - selling costs)

Typical IRR components:
  Cash flow: 3-6% per year
  Appreciation: 3-5% per year
  Mortgage paydown: 1-2% per year
  Tax benefits: 1-3% per year

Benchmarks:
  Excellent: > 20%
  Good:      15-20%
  Acceptable: 10-15%
  Below target: < 10%
```

## Depreciation (Tax Benefits)

```
Residential rental: 27.5-year straight-line
  Annual depreciation = (Building Value) / 27.5
  Note: Land is NOT depreciable (typically 20-30% of purchase price)

Commercial: 39-year straight-line
  Annual depreciation = (Building Value) / 39

Cost Segregation Study:
  Accelerates depreciation by reclassifying components
  5-year property: appliances, carpeting, cabinetry
  7-year property: furniture, fixtures
  15-year property: landscaping, sidewalks, parking lots
  Can generate 20-30% of building value in Year 1 deductions
```
